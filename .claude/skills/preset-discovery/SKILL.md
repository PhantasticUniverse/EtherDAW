## Usage Examples

### Simple Instrument
```json
{
  "lead": {
    "preset": "fm_epiano",
    "volume": -3
  }
}
```

### With Parameters
```json
{
  "bass": {
    "preset": "synth_bass",
    "params": {
      "brightness": 0.7,
      "warmth": 0.5,
      "punch": 0.8
    },
    "volume": -6
  }
}
```

### With Effects
```json
{
  "pad": {
    "preset": "warm_pad",
    "volume": -6,
    "effects": [
      { "type": "reverb", "wet": 0.5, "decay": 3.0 },
      { "type": "delay", "wet": 0.3, "time": "8n" }
    ]
  }
}
```

### Layered Sound
```json
{
  "supersaw_stack": {
    "layers": [
      { "preset": "sawtooth", "detune": -10 },
      { "preset": "sawtooth", "detune": 0 },
      { "preset": "sawtooth", "detune": 10 }
    ],
    "volume": -6
  }
}
```

## Semantic Parameters

| Parameter | Range | Description |
|-----------|-------|-------------|
| `brightness` | 0-1 | Tonal brightness/darkness |
| `warmth` | 0-1 | Analog warmth |
| `punch` | 0-1 | Attack transient strength |
| `richness` | 0-1 | Harmonic complexity |
| `attack` | 0-1 | Attack time (0=instant) |
| `decay` | 0-1 | Decay time |
| `sustain` | 0-1 | Sustain level |
| `release` | 0-1 | Release time |
